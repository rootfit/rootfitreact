### 기능 소개

- 회원가입
- 게시판
- 쇼핑몰
- 헬스리스트

### 팀원 및 역할

- 강민서: 회원가입과 로그인 기능 개발.
- 권보경: 회원전용 헬스리스트(투두리스트) 기능 개발, 팀장.
- 박종섭: 쇼핑몰 기능 개발.
- 신혜인: 게시판 기능 개발.
- 이지은: 회원전용 헬스리스트(투두리스트) 기능 개발
- 최지성: 게시판 기능 개발.
