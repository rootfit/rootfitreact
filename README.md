- 강민서 : 회원가입과 로그인 기능 개발, API총괄
- 권보경 : 회원전용 헬스리스트(투두리스트) 기능 개발, 팀장, 총괄 매니징
- 박종섭 : 쇼핑몰 기능 개발, DB&Design
- 신혜인 : 게시판 기능 개발, 스케줄 관리
- 이지은 : 회원전용 헬스리스트(투두리스트) 기능 개발, 프로젝트 매니징
- 최지성 : 게시판 기능 개발, 협업 유지보수
